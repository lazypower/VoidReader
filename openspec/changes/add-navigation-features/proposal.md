# Change: Add Navigation Features

## Why
Users need to efficiently navigate and search within documents. Find, outline navigation, and find-and-replace are essential features for any document reader/editor.

## What Changes
- Implement Find bar (Cmd+F) for searching document content
- Implement Find & Replace (Cmd+H) for edit mode
- Add toggleable Outline/TOC sidebar showing document headings
- Support keyboard navigation through search results and outline

## Impact
- Affected specs: navigation (new)
- Affected code: Search infrastructure, outline extraction, sidebar UI
